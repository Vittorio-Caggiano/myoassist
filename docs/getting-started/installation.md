# Installation Guide

This guide will walk you through the complete installation process for MyoAssist.

## Prerequisites

Before installing MyoAssist, ensure you have the following:

### System Requirements
- **Operating System**: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 18.04+)
- **Python**: Version 3.11 or higher
- **Memory**: Minimum 8GB RAM (16GB recommended)
- **Storage**: At least 5GB free space for models and assets

### Required Software
- **Python 3.11+**: Download from [python.org](https://python.org)
- **Git**: For cloning the repository
- **MuJoCo 3.1.5**: Physics engine (will be installed automatically)
